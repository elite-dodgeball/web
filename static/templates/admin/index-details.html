How to do Things

How to Add an Event

  1. Add a new Event or make sure it exists
    • Just leave the Count points attribute checked
    • Each region has a predetermined color for the Schedule
      • North: #04139A
      • East: #00BA21
      • South: #FF9300
      • West: #FF2600
      • Use the eyedropper tool in the color picker on the color blocks
  2. Add each division as a Division object
    • Match the Event id attribute to the Event from step one
    • The Count points attribute determines if the results are included in total point tallies
    • For Max teams, a negative value will mean there is no limit and a zero value will effectively mean it's invite-only
    • You shouldn't have to touch Current teams or Open invites at all
  3. Add each team as an Event team object
    • Match the Division id attribute to the Division in step two
    • The Place attribute can be left blank until the competition is over

How to Add a Team

  1. Every Region object should already exist
  2. Add a Team object
    • The Information isn't used at this time, but it's still required
    • While you can edit Wins, Losses, and Points, those attributes are automatically computed, so they're best left alone
    • The optimal dimensions for a Cover is 960x407, though anything with an equal ratio will do
    • The optimal dimensions for a Logo is 200x200, though anything with an equal ratio will do
  3. Add a Player object for each person on the team
    • Match the Team id attribute to the Team in step two
    • The optimal dimensions for a Headshot is 236x236, though anything with an equal ratio will do
    • The Bio attribute isn't used at this time and can be left blank

How to Add a Post

  1. Add a Post object
    • The optimal dimensions for a Header is 960x540, though anything with an equal ratio will do
    • The Blurb is the accompanying subheader text on the front page
    • Check the Featured attribute if you want to the post to be in the front page carousel
    • We use a WYSIWYG editor for the Body, but click the Source button in the top-right to get at the raw HTML

How to Commentate

  1. Avoid commentating on games you are playing in
  2. Dig up some facts on the competing teams beforehand and try to integrate them as quick tidbits
  3. Be descriptive in the action with location, pacing, context, etc.
  4. Make sure you know the names of all the players
    • Get them right when you call a play
    • Try to call out every player by name at least once in the video
    • As Richie Benaud says, "There are no teams in the world called 'we' or 'they'"
  5. Try to not repeat yourself and use exciting phrasing
    • Keep a list of common phrases or words and their alternatives
    • Throw: chunk, hurl, pelt. Catch: snag, corral, snatch. Hit: kill, tag, ding.
    • Mark them off as you go to keep up variety
    • Ditch bland, ineffective adjectives like "good," "amazing," and pretty much every adverb
  6. Don't ask superficially broad questions like "who do you think is going to win?"
    • Ask questions that have answers based on fact
    • Be productively analytical with some amount of foresight, not solely opinion
    • If you want to editorialize, channel it through crowd reactions

How to Not Ruin Things

  1. Don't ever delete any objects
  2. Upload images that look good, have proper dimensions, and look good
  3. Practice good copywriting
    • Check your grammar and spelling (try Hemingway or Grammarly)
    • Break up your sentences into easily consumable paragraphs
    • Use active wording and encourage the user to engage
    • Properly hyphenate your words (e.g., no-sting, round-robin, etc.)
  4. Optimize for SEO
    • Link back to the site where you can
    • Write new posts often and share them on Twiter and Facebook
    • Include links to our online presence in every YouTube description
    • Fill out every possible field in the advanced section of a YouTube video
